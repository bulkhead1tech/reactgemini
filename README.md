# Scam Detection Voice App

## Overview

The Scam Detection Voice App is designed to analyze audio inputs and detect potential scams in real-time. It leverages advanced machine learning algorithms to identify patterns and flag suspicious activities.

## Features

- **Real-time Detection:** Monitors audio feeds and provides immediate alerts for suspicious conversations.
- **User-friendly Interface:** Simple and intuitive design for ease of use.
- **Detailed Reporting:** Generates comprehensive reports on detected scams with timestamps and audio excerpts.

## Installation

To install the application, follow these steps:

1. Clone this repository:
   ```bash
   git clone https://github.com/bulkhead1tech/reactgemini.git
   ```
2. Navigate to the project directory:
   ```bash
   cd reactgemini
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage

Once the app is running:
- Speak into your device's microphone.
- The app will process your input and provide feedback on potential scams.
